MCP (Model Context Protocol)

LLM 工具调用的统一标准 — Agent 生态的 USB-C

🌍 现实案例：Anthropic 于 2024 年底发布了 MCP（Model Context Protocol）开放标准。Claude Desktop、Cursor、Windsurf、Amazon Q CLI 已支持 MCP，让 AI 能直接操作本地文件、Git、数据库等工具。
🌍 现实案例：MCP 的意义类似于 USB 统一了外设连接。之前每个 AI 应用要单独对接每个工具（N×M），MCP 变成 N+M——每个工具写一次 MCP server，所有 AI 都能用。
🌍 现实案例：GitHub 上已有 1000+ 个 MCP Server 实现，覆盖 Slack、Notion、PostgreSQL、AWS CLI 等常见工具。这是 AI Agent 生态爆发的基础设施——就像 App Store 对 iPhone 的意义。
